Tic Tac Toe Game

--Play and Enjoy--



You are X, your friend is O.
The first player to get 3 of the X or O marks in a row (up, down, across, or diagonally) is the winner. When all 9 squares are full, the game is over. If no player has 3 marks in a row, the game ends in a tie.
